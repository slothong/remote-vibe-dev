## 개요
원격 서버에 연결하여 TDD에 기반한 augmented coding을 지원하기 위한 UI를 제공한다.

## 시작 화면
- ssh host, port, username, key/password를 입력하는 폼을 표시한다.
- 입력된 정보로 ssh 연결에 성공하면 메인화면으로 이동한다.

## 메인 화면
- 원격 저장소에 ssh를 연결하고 터미널을 ui에 표시한다.
- 화면 상단에는 체크리스트를 표시한다. 체크리스트는 plan.md에 기반한 항목들이다.
- 체크리스트의 각 항목에는 go 버튼이 있다. 버튼을 누르면 go를 수행한다. 여기서 go는 claude의 /go 커맨드를 의미한다.
- 체크리스트의 각 항목에는 삭제 버튼이 있다. 버튼을 누르면 plan.md 파일에서 해당 항목이 삭제된다.
- 체크리스트의 각 섹션에는 항목 추가 폼이 존재한다. 버튼을 누르면 plan.md 파일에서 해당 항목이 추가된다.
- 화면 하단에는 터미널이 표시된다. 터미널을 열 때 `claude`를 실행한다.

## SSH 연결
- UI와 웹소켓으로 연결하고 원격 서버와 ssh 연결하여 브릿지 통신하는 서버를 구현해야 한다.
